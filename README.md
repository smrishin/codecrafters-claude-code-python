This is my attempt to the [Build Your own Claude Code Challenge](https://codecrafters.io/challenges/claude-code) by [CodeCrafters](https://codecrafters.io).

Claude Code is an AI coding assistant that uses Large Language Models (LLMs) to understand code and perform actions through tool calls. In this repo, we'll build our own Claude Code from scratch by implementing an LLM-powered coding assistant.
